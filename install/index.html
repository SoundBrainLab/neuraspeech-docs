Installation

This toolbox is built using primarily eelbrain. The following guide will walk you through how to install and set up the PRToolbox Python environment for the toolbx.

  1. Go to the Miniforge download page and download the installer for your system (Windows, macOS, or Linux) and install it.

  2. Download the Gitbub repository of the toolbox and unzip it.

  3. Inside, you should see a folder called "prptoolbox" and documents like README.md, requirements.yml, etc.

  4. Open a terminal or Anaconda Prompt, navigate to the directory of that folder (cd PATH/TO/THE/FOLDER), and then run:

mamba env create -f environment.yml
  1. This will create a Python environment called "prptoolbox" with required dependencies installed. You can activate this environment in the prompt or terminal using:
conda activate prptoolbox

If you're using a code editor like VS Code, you can choose the installed environment by selecting its name in the top right corner.
